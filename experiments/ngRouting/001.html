Angular Routing Experiment

About the experiment

In this experiment, Angular routing is used to traverse between different pages which are injected in the main page. Cick on the navigation link at the top of the page to traverse to different pages of the application.

Stock Market
  • Home
  • News
  • Portfolio

Code Snippet

                 var app = angular.module("StockApp", ["ngRoute"]);
                 app.config(['$routeProvider', function ($routeProvider) {
                    $routeProvider.
                     when('/home', {
                         templateUrl: 'stockApp/home.html'
                     }).
                     when('/news', {
                         templateUrl: 'stockApp/news.html'
                     }).
                    when('/portfolio', {
                        templateUrl: 'stockApp/portfolio.html'
                    }).
                     otherwise({
                         redirectTo: '/home'
                     });
                }]);

Code Snippet Explanation

Angular routing is used to route between different screens in the single page application. Routing controller decides which page to be injected in ng-view upon navigation menu click.
