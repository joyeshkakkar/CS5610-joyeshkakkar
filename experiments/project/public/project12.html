Project based Experiment

About the experiment

User can add an artist to favorite list. Profile page shows the favorite artists of the user and user can delete a favorite artist from profile page.
Project link

Code Snippet

                Client Side
                $scope.addArtistToFavorites = function (artist) {
                    var currentUser = ($rootScope.currentUser);
                    var artistToBeUpdated = { name: artist.name, id: artist.id };
                    currentUser.artists.push(artistToBeUpdated);
                    $http.put("/api/user/" + currentUser._id, currentUser)
                    .success(function (response) {
                        $scope.users = response;
                        alert("Artist \"" + artist.name + "\" is added to favorites.");
                    });
                }
                $scope.removeArtistFromFav = function (artist) {
                    var currentUser = ($rootScope.currentUser);
                    var index = $scope.favArtists.indexOf(artist);
                    $scope.favArtists.splice(index, 1);
                    currentUser.artists.splice(index, 1);
                    $http.put("/api/user/" + currentUser._id, currentUser)
                    .success(function (response) {
                        $scope.users = response;
                    });
                }

Code Snippet Explanation

Upon clicking the favorite button, artist is added to the database and shown as a favorite artist of the user. Favorite artists are shown on the profiles page and upon clicking the delete button, artist is removed from the favorites.
