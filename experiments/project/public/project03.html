About the experiment

In this experiment, I have used angular routing to navigate to different page of the application. Project can be accessed through link.
Project link

Code Snippet

                Client Side
                app.config(['$routeProvider', function ($routeProvider) {
                    $routeProvider.
                     when('/home', {
                         templateUrl: 'upbeat/home.html'
                         ,controller: 'HomeController'
                     }).
                     when('/about', {
                         templateUrl: 'upbeat/about.html'
                     }).
                    when('/profile', {
                        templateUrl: 'upbeat/profile.html'
                        , controller: 'ProfileController'
                    }).
                     otherwise({
                         redirectTo: '/home'
                     });
                }]);

Code Snippet Explanation

Routing is configured at client side and based on the configuration, different pages of application are opened.
