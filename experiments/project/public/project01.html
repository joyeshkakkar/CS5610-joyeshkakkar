About the experiment

In this experiment, I am using angular to fetch the track related data through the api and render the details in a tabular format. It can be accessed through link.

                    var title = $scope.searchByTitle;
                    $http.get("https://api.spotify.com/v1/search?q=" + title + "&type=track")
                    .success(function (response) {
                        console.log(response.tracks.items[0].artists[0].name);
                        $scope.tracks = response.tracks.items;            
                    })
                }

Code Snippet Explanation

On click of the Go button, http.get request is invoked on the api and json data is fetched from the request. Response is then shown in the tabular format.
