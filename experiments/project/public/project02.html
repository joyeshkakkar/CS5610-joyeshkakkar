Code Snippet

                Client Side
                $scope.removeTrack = function (track) {
                    var index = $scope.tracks.indexOf(track);
                    $scope.tracks.splice(index, 1);
                }

Code Snippet Explanation

On click of the X button, that particular row is removed from the view.
