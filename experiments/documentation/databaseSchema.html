Database Schema

I have used following collections in this project.

Track Schema

Specifies how favorite tracks in UserSchema and tracks in PlaylistSchema are saved. 
                            var TrackSchema = new mongoose.Schema({
                                name: String,
                                id: String,
                                preview_url: String,
                                external_url: String,
                                addedOn: { type: Date, default: Date.now }
                            }, { collection: "track" });

Album Schema

Specifies how favorite albums are saved in UserSchema. 
                             var AlbumSchema = new mongoose.Schema({
                                name: String,
                                id: String,
                                addedOn: { type: Date, default: Date.now }
                            }, { collection: "album" });

Artist Schema

Specifies how favorite artists are saved in UserSchema. 
                            var ArtistSchema = new mongoose.Schema({
                                name: String,
                                id: String,
                                addedOn: { type: Date, default: Date.now}
                            }, { collection: "artist" });

User Schema

Stores user's information and also favorite tracks, albums and artists. 
                            var UserSchema = new mongoose.Schema({
                                username: String,
                                password: String,
                                firstName: String,
                                lastName: String,
                                role: { type: String, default: 'Member' },
                                email: String,
                                phone: String,
                                tracks: [TrackSchema],
                                albums: [AlbumSchema],
                                artists: [ArtistSchema]
                            }, { collection: "user" });

Playlist Schema

Stores Playlists and tracks in it created by a user. 
                             var PlaylistSchema = new mongoose.Schema({
                                name: String,
                                tracks: [TrackSchema],
                                publish: { type: Boolean, default: 'false' },
                                userId: String,
                                addedOn: { type: Date, default: Date.now }
                            }, { collection: "playlist" });

Follower Schema

Stores playlist and user combination to determine which user follows which playlist. 
                            var FollowerSchema = new mongoose.Schema({
                                playlistId: String,
                                followerUserId: String
                            }, { collection: "follower" });
