Introduction

Upbeat is an attempt at making an interactive web application on MEAN stack using data fetched from web APIs. The application was created as the final project for CS 5610 Web Development course. It uses Spotify to fetch the data.
Code for this project has been open sourced at my Github repository for CS5610 course.

List of Available Features and Pages

Search
User can search for various tracks, albums and artists. Once logged in, user can add them to favorites and tracks to playlists.
Profile
User can view/remove his favorite tracks, albums and artists. User can add favorite tracks to playlists.
Playlist
Once loggedin, User can create/view his playlists and tracks in it. Published playlists of other users are also shown to enable user to follow.
Playlist follow/unfollow
Once loggedin,User can follow and unfollow the published playlists of other users.
Sign In
Enables user to sign in and access Profile and Playlist page. 'Adding to Favorites' functionality is also enabled once user logs in.
Sign Up
Enables user to create a new account.
Contact
Enables user to send an email to Upbeat Support for suggestions and complaints.



Technologies Used

Client Side

HTML
Pages are designed in HTML.
BootStrap
Used as CSS resource to enhance the look and feel of the project and also render it responsive.
JavaScript
Used to handle the interaction with the server and performing some validations.
Angular JS
Providing MVC architecture and handling dynamic data.

Server Side

NodeJS
Server side functionality handling.
Express(node modules)
Web framework for node.js
Mongoose
Used for mongoDB object modelling for node.js
NodeMailer
Used for sending emails using node.js

Database

MongoDB
Data storage
