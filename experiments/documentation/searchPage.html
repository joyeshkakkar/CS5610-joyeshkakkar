Search

Search page allows user to search various tracks, albums and artists. User has to click on the respective tabs to view tracks, albums and artists.This page can be accessed even if user does not login to the application. Although, upon logging in, user will have access to following functionalities:

  • Add/Remove tracks, albums, artists to favorites
  • Add tracks to already created playlists
