Playlist

Playlist page allows user to create/publish/unpublish a playlist. Also, user can follow/unfollow a published playlist by other user.
Followed playlists are shown in Followed Playlists tab.
User can only delete tracks from his own playlists.


Legend for various icons/buttons


User can listen to preview of the track.
User can listen to the track in spotify.
If it is next to playlist name, upon clicking, it remvoes the playlist. If it is next to track name, upon clicking, it removes the track from the playlist.
Click on this to publish a playlist. Once clicked, it changes to signifying that playlist is now published.
Click on this to unpublish a playlist. Once clicked, it changes to signifying that playlist is not published.
Follow a published playlist. If a user follows a playlist, it is removed from the published playlist list and is added to followed playlist list(accessed through Followed Playlists tab).
Unfollow a followed playlist.
