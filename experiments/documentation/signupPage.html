Sign Up

Sign up page allows user to create an account in the application.
User can sign up by choosing a username and password. In addition to that user also needs to provide following inputs before signing up.

  • First Name
  • Last Name
  • Email
  • Phone Number (optional)


Screenshot of signup page


Username should be unique across the application. So, if a user inputs a username which already exists, user is prompted to provide different username.
