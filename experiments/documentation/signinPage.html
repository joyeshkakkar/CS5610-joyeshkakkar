Sign In

Sign in page allows user to login to the application.
User can sign in by providing username and password. If a user provides incorrect credentials, a label specifying that is shown to user.

Upon signing in, user gets access to following functionalities.

  • Add tracks, albums, artists to favorites.
  • Create/view/publish playlists.
  • Add/remove tracks to/from playlists.
  • Follow/unfollow other user's published playlists.

If a user does not sign in, below functionalities are accessible.
  • Search for tracks, albums, artists.
  • Listen to tracks in spotify or a preview.
  • Contact the upbeat support through mail.
