Mongo DB Experiment

About the experiment

In this experiment, Node js is used to add a new record. First data is fetched using angular which calls a service at server side and then displays the data in tabluar format. For adding a row, angular calls another service at server side to add the data entered by user.

Code Snippet

                Client Side
                $scope.add = function (user) {
                    $http.post("/api/userMD", user)
                    .success(function (response) {
                        $scope.users = response;
                        $scope.user = null;
                    });
                };
                

                Server Side
                app.post("/api/userMD", function (req, res) {
                    var user1 = new UserModel(req.body);
                    user1.save(function (err, doc) {
                        UserModel.find(function (err, data) {
                            res.json(data);
                        });
                    });
                });

Code Snippet Explanation

Angular is used to add a new record to the mongodb. A function is called at client side on click of Add Button, which in turn invokes a post call at server and pushes the data.
