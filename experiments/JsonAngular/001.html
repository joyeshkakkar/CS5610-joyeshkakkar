AJAX UI Experiments

About the experiment

This experiment uses an API to fetch movie details and rendering the jsonp response on the client.

Input a movie title and get it's details.

Movie Details will be displayed here.

Code Snippet

            $(function () {
                $("#getMovie").click(function () {
                var movieTitle=$("#movieTitle").val();    
                $.ajax({
                        url: "http://www.myapifilms.com/imdb?title="+movieTitle+"&format=JSONP&aka=0&business=0&seasons=0&seasonYear=0
                                &technical=0&filter=N&exactFilter=0&limit=1&lang=en-us&actors=N&biography=0&trailer=0&uniqueName=0
                                &filmography=0&bornDied=0&starSign=0&actorActress=0&actorTrivia=0&movieTrivia=0&awards=0
                                &moviePhotos=N&movieVideos=N",
                        dataType: "jsonp",
                        success: renderMovies
                    });
                })
            });
            function renderMovies(movies) {
                var movie = movies[0];
                var title = movie.title;
                var plot = movie.plot;
                var urlPoster = movie.urlPoster;
                var h4 = $("&lth4>").append(title);
                var p= $("&ltp>").append(plot).append("&lt/hr>");
                var img=$("&ltimg>").attr("src", urlPoster);            
                $(".content").empty().append(h4).append(p).append(img);
            }

Code Snippet Explanation

API returns a response in JSONP format which is parsed and individual attributes of the object is captured and appended on the UI.
