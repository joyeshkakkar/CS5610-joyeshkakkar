JQuery UI Experiments

About the experiment

This experiment shows the draggable effect achieved using JQuery.

Below game images can be dragged horizontally.

Assassins Creed Unity

Call Of Duty

Crysis

Code Snippet

            $(function () {
                $(".acunity").draggable({
                    grid: [40, 40],
                    axis: "x"
                });
                $(".cod").draggable({
                    grid: [40, 40],
                    axis: "x"
                });
                $(".crysis").draggable({
                    grid: [40, 40],
                    axis: "x"
                });
            })

Code Snippet Explanation

JQuery objects are captured and made draggable. Also, objects can be dragged by grid size of [40,40] pixels and only in x-axis.
