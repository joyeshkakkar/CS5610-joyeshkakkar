JQuery UI Experiments

About the experiment

This experiment shows the sortable effect achieved using JQuery.

Below list can be reordered by clicking on a item and moving it up or down.

Game Titles

  • Assassins Creed Unity
  • Call Of Duty
  • Crysis
  • Halo
  • Hitman
  • DOTA
  • Counter Strike

Code Snippet

            $(function () {
                $(".reorderme").sortable({
                    axis: "y"
            })    
})

Code Snippet Explanation

JQuery objects are captured and made sortable. Also, objects can be reordered in y-axis only which is achieved by setting "axis" property.
