JQuery Experiments

About the experiment

This experiment shows the editable contnet effect achieved using JQuery.

User has an option to edit the content displayed upon clicking the game images. Edited content will not be saved as of now. I will extend the experiment to store the content in local storage.

Code Snippet

                $(".acunity").click(addContentACUnity);
                $(".cod").click(addContentCOD);
                $(".crysis").click(addContentCrysis);
                $("h1").click(function (event) {
                    console.log(event);
                });
                function addContentACUnity() {
                    $(".content").empty().html(
                            $("div").filter(".ACUnityText").attr("contenteditable", true));
                }
                function addContentCOD() {
                    $(".content").empty().html(
                                    $(".CODText").attr("contenteditable", true));
                }
                function addContentCrysis() {
                    $(".content").empty().html(
                                    $(".CrysisText").attr("contenteditable", true));
                }

Code Snippet Explanation

Here as the user will click on any of the game logos, JQuery will call a function and show related text in 'content' div. Also, contenteditable property is set to 'true' which gives user an option to edit the fields.

Assassins Creed Unity

Assassin's Creed is a historical fiction action-adventure open world stealth video game series. The game, which features up to four player co-op, a first for the series, is set in Paris on the eve of the French Revolution and follow Arno Dorian and his fellow assassins.

Call of Duty

Call of Duty (commonly shortened to CoD) is a first-person video game franchise. The series began on the PC, and later expanded to consoles and handhelds. Several spin-off games have also been released. The earlier games in the series are set primarily in World War II, including Call of Duty, Call of Duty 2, and Call of Duty 3.

Crysis

Crysis is a first-person shooter video game developed by Crytek in their headquarters in Frankfurt, Germany, published by Electronic Arts for Microsoft Windows and released in November 2007. It is the first game of a trilogy. A separate game entitled Crysis Warhead was released on September 16, 2008, and follows similar events as Crysis but from a different narrative perspective.
